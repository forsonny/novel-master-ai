This guide is for new users who want to start using Novel Master with minimal setup time.

It covers:
- [Requirements](/getting-started/quick-start/requirements): You will need Node.js and an AI model API Key.
- [Installation](/getting-started/quick-start/installation): How to Install Novel Master.
- [Configuration](/getting-started/quick-start/configuration-quick): Setting up your API Key, MCP, and more.
- [NRD](/getting-started/quick-start/prd-quick): Writing and parsing your first NRD (Novel Requirements Document).
- [Task Setup](/getting-started/quick-start/tasks-quick): Preparing your tasks for execution.
- [Executing Tasks](/getting-started/quick-start/execute-quick): Using Novel Master to execute tasks.
- [Rules & Context](/getting-started/quick-start/rules-quick): Learn how and why to build context in your project over time.

<Tip>
By the end of this guide, you'll have everything you need to begin working productively with Novel Master.
</Tip>
